NID QR Code Verification System

Scan a QR code to verify its authenticity.

Verification Endpoints:

Web Verification: /verify/<qr_id>
User-friendly web interface
API Verification: /api/verify/<qr_id>
JSON API for programmatic access

Admin & Analytics Endpoints:

Access Logs: /api/access_logs
Get all access logs (use ?qr_id=XXX to filter)
Access Statistics: /api/access_stats
Get comprehensive scan statistics
QR Codes List: /api/qr_codes
Get all QR codes with scan counts
